About NASS

Key resources, API access, and release schedule for USDA National Agricultural Statistics Service data.

📖 How to Use Quick Stats

  • Select Sector (Crops, Livestock, etc.)
  • Choose Commodity (Corn, Soybeans, etc.)
  • Pick Category (Yield, Area, Production)
  • Filter by Geography (State, County)
  • Set Year range and click Get Data
  • Export as CSV, Excel, or JSON

📡 NASS API Access

  • Free API at quickstats.nass.usda.gov/api
  • Free API key required — register at NASS
  • JSON, CSV, and XML output formats
  • Same parameters as the web query builder
  • Best for automated data pulls and scripts

📅 Key Data Releases

  • Crop Progress — Mondays, Apr–Nov, 4 PM ET
  • WASDE — Monthly (see calendar)
  • Prospective Plantings — Late March
  • Acreage Report — Late June
  • Grain Stocks — Quarterly
  • Census of Agriculture — Every 5 years
