REST API
Data Endpoints
Direct JSON access to AGSIST data. Static files on a CDN ‚Äî fast, reliable, no auth. Suitable for apps, spreadsheets, research tools, and automation.
GET
https://agsist.com/api/v1/daily/current.json
Today's AGSIST Daily briefing ‚Äî headline, lead, all sections, publish timestamp. ~3 KB.
GET
https://agsist.com/api/v1/daily/YYYY-MM-DD.json
Archived briefing by date. Replace YYYY-MM-DD with any past weekday. Available from launch date forward.
GET
https://agsist.com/api/v1/prices/current.json
Latest commodity prices ‚Äî corn, beans, wheat, milk, cattle, hogs. 15-min delayed. ~2 KB.
GET
https://agsist.com/api/v1/spray/current.json
Spray conditions by lat/lon. Pass ?lat=44.5&lon=-93.2 for location-specific results.
GET
https://agsist.com/api/v1/urea/current.json
Urea volatilization risk score for a lat/lon. Pass ?lat=44.5&lon=-93.2. Returns score, level, factor breakdown. ~2 KB.
GET
https://agsist.com/api/v1/spray/current.json
Spray advisory for a lat/lon. Returns overall rating, wind/temp/humidity/inversion factor scores, and hourly window forecast. ~3 KB.
GET
https://farmers1st.com/api/v3/current.json
FFAI ‚Äî Farmers First Ag Index. Current quarter composite, sub-indexes, regime, outlook. CC BY 4.0.

‚öôÔ∏è JavaScript SDK Reference
geo.js ‚Äî Shared Utility Functions

Including geo.js exposes global utilities used by all AGSIST pages.

fetchStooqSym(entry, callback)
Fetches a single CBOT futures price from Stooq with auto-fallback proxy.
const entry = STOOQ_BY_LABEL['Corn (front)'];
fetchStooqSym(entry, (err, result) => {
  if (result) console.log(result.close); // e.g. "4.3825"
});
STOOQ_BY_LABEL[label]
Lookup table of all available Stooq symbols. Keys match geo.js STOOQ_QUOTES labels.
// Available labels:
'Corn (front)', "Corn Dec'26"
'Beans (front)', "Beans Nov'26"
'Wheat (front)', 'Live Cattle'
'Lean Hogs', 'Dollar Index'
'10-Yr Treasury', 'Gold'
AGSIST_GEO.getState() / setState()
Read/write the user's selected state (persists via localStorage). From state.js.
// Include state.js first
const st = AGSIST_GEO.getState(); // 'WI'
AGSIST_GEO.setState('MN');
// Fires 'agsist-state-change' event
AGSIST_GEO.detectLocation(ok, fail)
Browser geolocation ‚Üí Nominatim reverse-geocode ‚Üí 2-letter state abbreviation.
AGSIST_GEO.detectLocation(
  abbr => console.log('State:', abbr),
  ()   => console.log('Location unavailable')
);
